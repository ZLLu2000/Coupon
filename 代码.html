三、SVM-ANN混合预测模型¶

1、模型解释¶

本文选取"destination","coupon","expiration","Bar","CoffeeHouse","toCoupon_GEQ25min"等字段作为输入，Y字段作为输出；首先使用sklearn模块分别训练SVM和ANN模型，然后根据其预测结果训练梯度提升树（GradientBoostingClassifier）以产生最终输出。关键步骤是从测试集上的基本模型生成预测，然后使用这些预测作为输入特征来训练元模型。

2、SVM-ANN混合模型¶

In [67]:
from sklearn.model_selection import train_test_split
import warnings
warnings.filterwarnings('ignore')

#划分数据集
X = df[["destination","coupon","expiration","Bar","CoffeeHouse","toCoupon_GEQ25min"]]#自变量
Y = df[["Y"]]#因变量
X_train,X_test,Y_train,Y_test = train_test_split(X,Y,test_size=0.3)

#建立SVM和ANN模型
from sklearn.svm import SVC
from sklearn.neural_network import MLPClassifier
svm = SVC(C=1000,kernel="rbf")
ann = MLPClassifier(hidden_layer_sizes=100,activation="tanh",solver="lbfgs",batch_size=64)

#训练模型
svm.fit(X_train,Y_train)
ann.fit(X_train,Y_train)
#预测
svm_pre = svm.predict(X_test)
ann_pre = ann.predict(X_test)

#模型混合
blender = np.vstack((svm_pre,ann_pre)).T
blender_target = Y_test
from sklearn.ensemble import GradientBoostingClassifier
gb = GradientBoostingClassifier(n_estimators=20)#迭代次数10
gb.fit(blender,blender_target)
final_pre = gb.predict(blender)
